# MoviesMemo

Personal website to track movies watched.


- Searching movies by Id and Name
- Ability to watch and store watched movie

# Tech stack:
### Frontend
   - ViteJS
   - ReactJS
   - Typescript
   - MUI
### API
   - TMDB
### Backend
   - NodeJS
### Database
   - Firebase
